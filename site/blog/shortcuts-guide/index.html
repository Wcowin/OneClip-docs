快捷键使用指南¶

OneClip 提供丰富的快捷键，熟练使用可以大幅提升效率。

默认快捷键¶

OneClip 特有快捷键¶

快捷键 功能
Cmd + Shift + V 打开剪贴板历史
Cmd + Shift + F 搜索剪贴板历史
Cmd + Shift + S 保存到收藏

历史记录窗口操作¶

快捷键 功能
↑ / ↓ 上下选择历史记录
Enter 粘贴选中项
Esc 关闭窗口
Cmd + F 搜索当前列表
Cmd + Delete 删除选中项

自定义快捷键¶

OneClip 支持自定义快捷键，满足个人使用习惯。

设置步骤¶

  1. 打开 OneClip 设置
  2. 选择「快捷键」选项卡
  3. 点击要修改的快捷键
  4. 按下新的快捷键组合
  5. 保存设置

推荐快捷键¶

根据使用习惯，推荐以下快捷键：

  • Cmd + \：打开剪贴板历史（避免与系统快捷键冲突）
  • Cmd + Option + V：粘贴历史记录
  • Cmd + Option + F：搜索剪贴板

快捷键使用技巧¶

1. 快速访问历史记录¶

使用 Cmd + Shift + V 快速打开剪贴板历史，然后：

  • 使用方向键浏览
  • 输入文字快速搜索
  • 按 Enter 粘贴选中项

2. 搜索历史记录¶

在历史记录窗口中：

  • 按 Cmd + F 进入搜索模式
  • 输入关键词搜索
  • 使用方向键选择结果
  • 按 Enter 粘贴

3. 收藏常用内容¶

选中内容后按 Cmd + Shift + S 保存到收藏，便于快速访问。

4. 删除不需要的记录¶

选中不需要的历史记录，按 Cmd + Delete 删除。

快捷键冲突处理¶

常见冲突¶

OneClip 的默认快捷键可能与以下应用冲突：

  • 截图工具：Cmd + Shift + 4（部分截图工具）
  • Finder：Cmd + Shift + F（查找）
  • 浏览器：Cmd + Shift + F（查找）

解决方案¶

如果快捷键冲突，可以：

  1. 修改 OneClip 快捷键
  2. 打开 OneClip 设置
  3. 选择「快捷键」选项卡

  4. 修改冲突的快捷键

  5. 修改其他应用快捷键

  6. 在冲突应用的设置中修改快捷键

  7. 使用不同的快捷键组合

  8. 尝试使用 Cmd + Option 组合
  9. 使用 Cmd + Control 组合

常见问题¶

快捷键不生效怎么办？¶

  1. 检查快捷键设置是否正确
  2. 重启 OneClip 应用
  3. 检查是否有其他应用占用快捷键
  4. 尝试修改为其他快捷键组合

如何查看所有快捷键？¶

在 OneClip 设置中：

  1. 打开 OneClip 设置
  2. 选择「快捷键」选项卡
  3. 查看所有可用的快捷键

如何重置快捷键？¶

在 OneClip 设置中：

  1. 打开 OneClip 设置
  2. 选择「快捷键」选项卡
  3. 点击「重置为默认」按钮

实用场景¶

场景1：快速复制粘贴代码¶

  1. 复制代码
  2. 按 Cmd + Shift + V 打开剪贴板历史
  3. 使用方向键找到代码
  4. 按 Enter 粘贴

场景2：快速回复邮件¶

  1. 创建邮件模板并保存到收藏
  2. 按 Cmd + Shift + V 打开剪贴板历史
  3. 使用 Cmd + F 搜索邮件模板
  4. 按 Enter 粘贴

场景3：快速查找链接¶

  1. 复制链接
  2. 按 Cmd + Shift + S 保存到收藏
  3. 下次使用时按 Cmd + Shift + V 打开历史
  4. 使用 Cmd + F 搜索链接
  5. 按 Enter 粘贴

注意事项¶

系统快捷键¶

以下快捷键是 macOS 系统快捷键，不建议修改：

  • Cmd + C：复制
  • Cmd + V：粘贴
  • Cmd + X：剪切
  • Cmd + A：全选
  • Cmd + Z：撤销
  • Cmd + Shift + Z：重做

应用快捷键¶

某些快捷键可能被其他应用占用：

  • Cmd + Space：Spotlight 搜索
  • Cmd + Tab：切换应用
  • Cmd + Option + Esc：强制退出应用

总结¶

熟练使用快捷键可以大幅提升效率：

  • 减少鼠标操作
  • 提高操作速度
  • 提升工作效率

OneClip 提供丰富的快捷键和自定义功能，满足不同用户的需求。
