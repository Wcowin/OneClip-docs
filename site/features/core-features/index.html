核心功能¶

OneClip 是 Mac 上专业的剪贴板管理工具，支持无限历史记录、智能搜索、AI 本地模型、粘贴板堆栈、拖拽容器等强大功能。

📋 剪贴板历史记录¶

自动监控¶

OneClip 持续监控系统剪贴板变化，自动保存复制的内容：

  • 实时监控：自动检测剪贴板变化
  • 智能去重：自动识别并合并重复内容
  • 完整元数据：保留来源应用、时间戳、文件路径等
  • 容量管理：自动清理超过保留期限的内容
  • 收藏保护：收藏的内容永久保存

支持的内容类型¶

类型 格式
📝 文本 纯文本、富文本、代码
🖼️ 图片 PNG、JPEG、GIF、WebP、HEIC 等
📁 文件 所有文件类型
🎬 视频 MP4、MOV 等
🎵 音频 MP3、M4A 等
📄 文档 PDF、Word、Excel 等
📦 压缩包 ZIP、RAR 等

快速操作¶

  • 点击 - 复制到剪贴板
  • 空格 - 预览内容
  • 双击 - 编辑文本内容
  • 右键 - 显示完整菜单
  • 拖拽 - 拖拽到其他应用

🔍 智能搜索¶

多维度搜索¶

  • 内容搜索：搜索文本内容、文件名
  • 来源搜索：按复制来源应用筛选
  • 时间搜索：按时间范围查找
  • 类型筛选：按内容类型分类
  • 模糊匹配：智能容错，不需要精确输入

搜索技巧¶

  • 实时搜索：输入即搜索，无需按回车
  • 组合筛选：搜索 + 分类同时使用
  • 搜索历史：自动记住常用搜索词

🗂️ 智能分类¶

系统分类（自动）¶

  • 📋 全部 - 所有历史记录
  • ⭐ 收藏 - 标记为收藏的项目
  • 📝 文本 - 纯文本和富文本
  • 🖼️ 图片 - 所有图片格式
  • 📁 文件 - 文件和文件夹
  • 🎬 视频 - 视频文件
  • 🎵 音频 - 音频文件
  • 📄 文档 - PDF、Office 等
  • 💻 代码 - 代码文件

自定义分类¶

创建您专属的分类规则：

  1. 打开设置 → 分类管理
  2. 点击"新建分类"
  3. 设置分类名称和图标
  4. 添加匹配规则（内容包含、来源应用、文件扩展名等）

⌨️ 全局快捷键¶

📥 栈粘贴板¶

栈粘贴板是一个特殊的剪贴板管理工具，按照先进后出（LIFO）的顺序管理多个内容。

使用场景¶

  • 批量填写表单时，依次粘贴多个字段
  • 整理文档时，按顺序粘贴多段内容
  • 编程时，依次粘贴多行代码
  • 数据录入时，快速粘贴多个数据项

操作方法¶

  • 添加到栈：按 ⌘ + ⇧ + C 将当前剪贴板内容添加到栈顶
  • 依次粘贴：按 ⌘ + V 从栈顶依次取出并粘贴内容
  • 查看栈内容：通过栈粘贴板窗口查看和管理所有内容

提示

栈粘贴板采用"后进先出"的顺序，最后添加的内容会最先被粘贴。

🎯 拖拽容器¶

拖拽容器是一个临时暂存区，可以收集多个项目后统一处理。

使用场景¶

  • 收集多个文件准备批量发送
  • 暂存稍后需要用到的内容
  • 整理素材和资料

操作方法¶

  • 打开容器：按 ⌘ + ⇧ + D 呼出拖拽容器窗口
  • 添加内容：将文件/图片/文本拖拽到拖拽容器窗口
  • 使用内容：从容器中拖出项目到目标应用
  • 管理内容：可以删除单个项目或一键清空容器

⚡ 快捷回复¶

预设常用文本，一键快速输入，提升沟通效率。

使用方法¶

  1. 按 ⌘ + ⇧ + R 打开快捷回复窗口
  2. 选择或搜索需要的模板
  3. 点击或按回车即可复制到剪贴板

支持的内容类型¶

  • 📝 纯文本 - 常用回复、邮件模板
  • 🖼️ 图片 - 常用图片、表情包
  • 📁 文件 - 常用文档、附件

⭐ 收藏功能¶

将重要内容标记为收藏，确保不会被自动清理。

  • 添加收藏：点击项目右侧的星标图标
  • 取消收藏：再次点击星标图标
  • 查看收藏：点击左侧"收藏"分类

收藏特性：

  • ✅ 永久保存，不受自动清理影响
  • ✅ 支持云同步（如启用）
  • ✅ 独立备份和导出
  • ✅ 快速访问和搜索

👁️ 内容预览¶

按 空格键 可以快速预览任何内容：

  • 文本预览：Markdown 格式化显示
  • 图片预览：高清大图展示
  • 文件预览：Quick Look 系统预览
  • 代码预览：语法高亮显示
  • 视频预览：内联播放

预览技巧

  • 预览窗口支持左右箭头键快速切换
  • 支持多文件预览（如选中多个文件）
  • 图片预览支持缩放和旋转

✏️ 编辑功能¶

双击文本项目可以打开编辑器，支持：

  • 实时预览：Markdown 渲染
  • 语法高亮：代码高亮显示
  • 查找替换：快速定位和替换
  • 撤销重做：支持多步撤销
  • 自动保存：编辑内容自动保存

编辑器快捷键¶

功能 快捷键
保存 ⌘ + S
查找 ⌘ + F
查找下一个 ⌘ + G
替换 ⌘ + H
撤销 ⌘ + Z
重做 ⌘ + ⇧ + Z

📦 批量操作¶

批量操作功能让您可以同时处理多个项目：

  1. 点击主界面左上角的"批量操作"按钮
  2. 选择需要操作的项目（点击即可选中）
  3. 使用底部工具栏进行操作：
  4. 批量删除
  5. 批量收藏
  6. 批量取消收藏
  7. 批量导出

🏷️ 标题功能¶

为剪贴板项目添加自定义标题，方便组织和查找：

  1. 找到需要添加标题的项目
  2. 右键点击 → "编辑标题"
  3. 输入有意义的标题（如"项目需求文档"）
  4. 保存后标题会显示在项目上方

标题优势：

  • ✅ 更容易识别重要内容
  • ✅ 支持标题搜索
  • ✅ 云同步时保留标题
  • ✅ 提升内容组织效率

📊 菜单栏功能¶

OneClip 的菜单栏图标提供快速访问：

  • 最近项目：显示最近 10 个剪贴板项目
  • 一键粘贴：点击项目直接复制到剪贴板
  • 悬停预览：鼠标悬停显示内容预览
  • 快速访问：打开主窗口、栈粘贴板、拖拽容器、快捷回复、设置等

⚙️ 高级设置¶

常规设置¶

  • 启动设置：开机自启、登录后启动
  • 外观设置：Dock 图标、菜单栏图标显示
  • 语言设置：界面语言切换
  • AI 功能：支持云端 AI（智谱 AI）和本地 AI（Ollama）

监控设置¶

  • 监控间隔：调整剪贴板检查频率（0.1-2.0秒）
  • 排除应用：设置不监控的应用（如密码管理器）
  • 内容过滤：设置不记录的内容类型

存储设置¶

  • 保留期限：设置历史记录保存天数
  • 最大数量：限制历史记录数量
  • 缓存管理：清理图片和文件缓存
  • 存储位置：查看和管理数据存储

快捷键设置¶

  • 主窗口：自定义打开主窗口的快捷键
  • 快捷回复：自定义快捷回复窗口快捷键
  • 冲突检测：自动检测与系统或其他应用的冲突
