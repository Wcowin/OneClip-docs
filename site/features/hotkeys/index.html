快捷键设置¶

OneClip 提供丰富的快捷键支持，所有快捷键均可自定义。

核心快捷键¶

功能 快捷键 说明
打开主窗口 ⌘ + ⇧ + V 显示/隐藏主窗口
打开快捷回复 ⌘ + ⇧ + R 打开快捷回复窗口
呼出栈粘贴板 ⌘ + ⇧ + C 打开栈粘贴板窗口
依次粘贴栈内容 ⌘ + V 从栈顶依次取出并粘贴
呼出拖拽容器 ⌘ + ⇧ + D 打开拖拽容器窗口
搜索 ⌘ + F 在主窗口中搜索
预览 Space 预览选中内容
关闭窗口 Esc 隐藏当前窗口

界面内快捷键¶

功能 快捷键 说明
向上/下导航 ↑ / ↓ 选择项目
确认选择 Enter 粘贴选中内容
删除项目 Delete 删除选中项目
收藏/取消收藏 ⌘ + S 切换收藏状态

编辑器快捷键¶

功能 快捷键
保存 ⌘ + S
查找 ⌘ + F
查找下一个 ⌘ + G
替换 ⌘ + H
撤销 ⌘ + Z
重做 ⌘ + Shift + Z

自定义快捷键¶

设置步骤¶

  1. 打开 OneClip 设置（⌘ + ,）
  2. 选择"快捷键"标签页
  3. 点击要修改的快捷键输入框
  4. 按下新的快捷键组合
  5. 设置自动保存并生效

修饰键说明¶

  • Command (⌘): 主要修饰键
  • Option (⌥): 备选修饰键
  • Shift (⇧): 大小写修饰键
  • Control (⌃): 控制修饰键

提示

推荐使用双修饰键组合，如 ⌘ + ⇧ 避免与系统快捷键冲突。

权限设置¶

辅助功能权限¶

OneClip 需要辅助功能权限才能使用全局快捷键：

  1. 打开系统设置 → 隐私与安全性
  2. 选择"辅助功能"
  3. 添加 OneClip 并启用
  4. 重启应用使权限生效

故障排除¶

快捷键不响应¶

检查清单：

  • ✅ 辅助功能权限已开启
  • ✅ OneClip 正在运行
  • ✅ 快捷键没有冲突
  • ✅ 没有被其他应用占用

解决方案：

  1. 重新设置辅助功能权限
  2. 更改为不同的快捷键组合
  3. 重启 OneClip 应用

避免冲突¶

  • 建议使用 Shift 或 Control 作为辅助修饰键
  • 避免使用常见的系统快捷键（如 ⌘ + C/V）
  • OneClip 会自动检测并提示冲突
