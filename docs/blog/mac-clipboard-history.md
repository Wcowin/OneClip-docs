# Mac剪贴板历史记录

Mac 的默认剪贴板只能保存一条记录，如何管理和查看剪贴板历史记录？

## Mac 默认剪贴板的限制

Mac 的默认剪贴板有以下限制：

- 只能保存一条记录
- 复制新内容会覆盖旧内容
- 无法查看历史记录
- 重启后内容丢失

## 解决方案

### 方案1：使用 OneClip

OneClip 是专为 Mac 设计的专业剪贴板管理工具：

**特点**：
- 无限历史记录
- 强大的搜索功能
- 跨设备云同步
- 端到端加密
- 简洁美观的界面

**使用方法**：
1. 下载安装 OneClip
2. 启动应用
3. 自动记录剪贴板历史
4. 使用快捷键 `Cmd + Shift + V` 打开历史记录

### 方案2：使用终端命令

Mac 可以通过终端查看剪贴板历史：

```bash
# 查看当前剪贴板内容
pbpaste

# 复制内容到剪贴板
echo "内容" | pbcopy
```

**局限性**：
- 只能查看当前内容
- 无法查看历史记录
- 不支持搜索

### 方案3：使用其他工具

除了 OneClip，还有其他选择：

- **Paste**：功能丰富，但价格较高
- **CopyClip**：轻量级，但功能简单
- **原生剪贴板**：系统自带，但功能有限
