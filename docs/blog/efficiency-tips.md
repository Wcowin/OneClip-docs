# 效率提升技巧

使用剪贴板管理工具可以大幅提高工作效率。本文分享一些实用技巧。

## 1. 善用分类管理

将剪贴板内容按类别整理，便于快速查找：

- **代码片段**：常用的代码模板
- **文本模板**：常用的回复、邮件模板
- **链接收藏**：常用的网址链接
- **图片素材**：常用的图片资源

## 2. 使用快捷键

OneClip 提供丰富的快捷键，熟练使用可以大幅提升效率：

- `Cmd + Shift + V`：打开剪贴板历史
- `Cmd + Shift + C`：复制到剪贴板
- `Cmd + Shift + X`：剪切到剪贴板
- `Cmd + Shift + F`：搜索剪贴板

## 3. 设置自动清理

定期清理旧的剪贴板记录，保持剪贴板整洁：

- 设置保留天数
- 设置最大记录数量
- 定期手动清理

## 4. 使用搜索功能

快速查找历史记录：

- 支持模糊搜索
- 支持正则表达式
- 按类型筛选

## 5. 跨设备同步

在不同设备间共享剪贴板内容：

- 端到端加密保护隐私
- 实时同步
- 支持多设备

## 6. 收藏常用内容

将常用的内容标记为收藏：

- 一键访问
- 优先显示
- 便于管理

## 7. 使用文本模板

创建常用的文本模板：

- 邮件回复模板
- 代码注释模板
- 文档模板

## 8. 定期备份

定期备份剪贴板数据：

- 防止数据丢失
- 便于恢复
- 数据安全
