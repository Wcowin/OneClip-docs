# 支付说明

OneClip 支持多种支付方式，安全便捷。

## 支付方式

### 支付宝扫码支付

我们推荐使用支付宝扫码支付，这是最安全、最便捷的支付方式。

**支付流程**：

1. 在购买页面点击「立即购买」按钮
2. 系统会生成支付宝二维码
3. 使用支付宝 App 扫描二维码
4. 确认支付金额和订单信息
5. 完成支付

### 支付安全

- 采用业界标准加密技术保护支付信息
- 支付过程完全在支付宝平台完成
- 不存储任何支付敏感信息

## 支付流程

### 1. 选择版本

根据您的需求选择合适的版本：
- 月度版：¥5/月
- 年度版：¥50/年
- 终身版：¥29.90/终身

### 2. 确认订单

确认购买版本和支付金额，检查订单信息是否正确。

### 3. 完成支付

使用支付宝扫码支付，支付成功后会自动跳转到成功页面。

### 4. 获取许可证

支付成功后，系统会自动生成许可证密钥并发送到您的邮箱。

### 5. 激活使用

在 OneClip 设置中输入许可证密钥，即可立即使用高级功能。

## 发票

如需要发票，请在支付后联系客服，提供以下信息：
- 订单号
- 发票抬头
- 发票金额
- 发票类型（普通发票/专用发票）

## 退款政策

### 退款条件

- 购买后 7 天内可申请全额退款
- 因技术问题无法使用可申请退款
- 终身版提供 7 天无理由退款保障

### 退款流程

1. 联系客服说明退款原因
2. 提供订单号和许可证密钥
3. 客服审核通过后，原路退回
4. 退款周期：3-7 个工作日

### 退款限制

- 已使用的许可证密钥无法退款
- 超过 7 天的订单不支持退款
- 恶意退款将影响后续购买

## 支付失败处理

如果支付失败，请检查以下几点：

### 支付宝余额不足

- 请确保支付宝账户有足够的余额
- 或绑定银行卡进行支付

### 网络连接问题

- 确保设备已连接互联网
- 检查网络连接是否稳定
- 尝试切换网络环境

### 支付限额

- 检查支付宝的支付限额设置
- 联系支付宝客服提高限额
