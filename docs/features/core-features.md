# 核心功能

OneClip 是 Mac 上专业的剪贴板管理工具，支持无限历史记录、智能搜索、AI 本地模型、粘贴板堆栈、拖拽容器等强大功能。

## 📋 剪贴板历史记录

### 自动监控

OneClip 持续监控系统剪贴板变化，自动保存复制的内容：

- **实时监控**：自动检测剪贴板变化
- **智能去重**：自动识别并合并重复内容
- **完整元数据**：保留来源应用、时间戳、文件路径等
- **容量管理**：自动清理超过保留期限的内容
- **收藏保护**：收藏的内容永久保存

### 支持的内容类型

| 类型 | 格式 |
|------|------|
| 📝 文本 | 纯文本、富文本、代码 |
| 🖼️ 图片 | PNG、JPEG、GIF、WebP、HEIC 等 |
| 📁 文件 | 所有文件类型 |
| 🎬 视频 | MP4、MOV 等 |
| 🎵 音频 | MP3、M4A 等 |
| 📄 文档 | PDF、Word、Excel 等 |
| 📦 压缩包 | ZIP、RAR 等 |

### 快速操作

- `点击` - 复制到剪贴板
- `空格` - 预览内容
- `双击` - 编辑文本内容
- `右键` - 显示完整菜单
- `拖拽` - 拖拽到其他应用

## 📥 栈粘贴板

栈粘贴板是一个特殊的剪贴板管理工具，按照先进后出（LIFO）的顺序管理多个内容。

### 使用场景

- 批量填写表单时，依次粘贴多个字段
- 整理文档时，按顺序粘贴多段内容
- 编程时，依次粘贴多行代码
- 数据录入时，快速粘贴多个数据项

### 操作方法

- **添加到栈**：按 `⌘ + ⇧ + C` 将当前剪贴板内容添加到栈顶
- **依次粘贴**：按 `⌘ + V` 从栈顶依次取出并粘贴内容
- **查看栈内容**：通过栈粘贴板窗口查看和管理所有内容

!!! tip "提示"
    栈粘贴板采用"后进先出"的顺序，最后添加的内容会最先被粘贴。

## 🎯 拖拽容器

拖拽容器是一个临时暂存区，可以收集多个项目后统一处理。

### 使用场景

- 收集多个文件准备批量发送
- 暂存稍后需要用到的内容
- 整理素材和资料

### 操作方法

- **打开容器**：按 `⌘ + ⇧ + D` 呼出拖拽容器窗口
- **添加内容**：将文件/图片/文本拖拽到拖拽容器窗口
- **使用内容**：从容器中拖出项目到目标应用
- **管理内容**：可以删除单个项目或一键清空容器

## ⭐ 收藏功能

将重要内容标记为收藏，确保不会被自动清理。

- **添加收藏**：点击项目右侧的星标图标
- **取消收藏**：再次点击星标图标
- **查看收藏**：点击左侧"收藏"分类

收藏特性：

- ✅ 永久保存，不受自动清理影响
- ✅ 支持云同步（如启用）
- ✅ 独立备份和导出
- ✅ 快速访问和搜索

## 🌐 局域网同步

在同一 WiFi 网络下，实现 Mac 与手机/Mac 之间的实时双向同步。

### 特点

- **无需安装 App**：手机端通过浏览器即可使用
- **数据安全**：仅在局域网内传输，不经过互联网
- **实时同步**：剪贴板内容实时同步
- **支持格式**：文本、图片等多种格式

### 使用方法

1. 在 OneClip 设置中开启局域网同步
2. 手机扫描显示的二维码
3. 在浏览器中即可查看和同步剪贴板内容
