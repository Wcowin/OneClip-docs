## 获取更新

### 自动更新

OneClip 支持 Sparkle 自动检查更新：

- 应用会定期检查新版本
- 有新版本时会提示更新
- 支持后台静默下载

### 手动更新

**Homebrew 用户：**

```bash
brew upgrade --cask oneclip
```

**手动安装用户：**

1. 访问 [GitHub Releases](https://github.com/Wcowin/OneClip/releases)
2. 下载最新版本
3. 替换现有应用
